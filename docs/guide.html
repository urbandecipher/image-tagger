ImageTag Studio 是一個完全本地運行的圖片打標工具，
無需網路、無需 API Key，保護你的隱私。

01 · 安裝

安裝與啟動

首次設定只需幾分鐘，之後雙擊即可開啟。

1

下載專案

從 GitHub 下載 ZIP 並解壓縮，或使用 git clone。

Terminal
git clone https://github.com/your-username/image-tagger.git
cd image-tagger
2

建立虛擬環境並安裝依賴

建議使用虛擬環境隔離套件，避免與其他 Python 專案衝突。

Terminal
python -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
3

啟動應用程式

雙擊 start.bat，或在終端機執行以下指令。首次啟動會自動下載 AI 模型（約 200 MB）。

Terminal
.\start.bat
💡

啟動後瀏覽器會自動開啟 http://localhost:8000，之後每次雙擊 start.bat 即可。

4

（選用）打包為 .exe

如果希望雙擊 exe 直接啟動，不需要開終端機：

Terminal
pip install pyinstaller
python build.py
# 完成後在 dist/ 資料夾找到 ImageTagStudio.exe
02 · 使用

基本使用流程

三個步驟完成從掃描到搜索的完整工作流程。

①

掃描資料夾

在左側「圖片資料夾」欄位輸入你的圖片路徑（支援子資料夾遞迴掃描），點「掃描資料夾」。系統會找出所有 JPG、PNG、WebP、GIF、BMP 圖片並建立索引。

②

AI 自動打標

調整信心閾值（預設 0.35），點「開始打標」。AI 會分析每張圖片並生成 tag，進度條顯示完成狀況。打標完成後 tag 永久儲存，重新啟動不會遺失。

⚡

信心閾值說明：數值越高，tag 越少但越精準；數值越低，tag 越多但可能有噪音。建議從 0.35 開始，視結果調整。

③

搜索與瀏覽

打標完成後，左側會顯示分類 Tag 面板，點選 tag 即可篩選。也可以在搜索框輸入關鍵字，支援英文和中文搜索。

⚠️

注意：已打標的圖片不會重複處理。若需重新打標，可在 Detail 面板手動編輯，或刪除 tags.db 重新開始。

03 · 功能

介面功能說明

各區域的功能一覽。

🏷️

Tag 分類面板

左側 Tag 以人物、髮型、表情、服裝、場景、畫風等分類展示，支援一鍵全收合/全展開。

🔍

智慧搜索

輸入時自動彈出建議，支援中英文輸入，逗號分隔多個條件進行組合搜索。

✏️

手動編輯 Tag

點擊任意圖片開啟 Detail 面板，可新增、刪除 tag，支援輸入中文自動轉換。

📂

開啟原始位置

Detail 面板中點擊 📂 按鈕，直接在 Windows 檔案總管中選中並顯示該圖片。

📄

分頁瀏覽

底部顯示頁碼按鈕，可直接點選跳頁，或使用「跳至」輸入框快速前往指定頁。

🌙

亮/暗主題

右上角點擊 🌙/☀️ 切換主題，偏好自動記憶，中英文介面同樣支援切換。

05 · FAQ

常見問題

遇到問題先看這裡。

啟動後瀏覽器沒有自動開啟？ ›
手動開啟瀏覽器並輸入 http://localhost:8000。如果顯示無法連線，請確認終端機視窗中伺服器是否已成功啟動（看到「WD-Tagger 模型載入完成」字樣）。
打標速度很慢？ ›
目前使用 CPU 模式。若有 NVIDIA GPU，可安裝 CUDA 12.x 並重新安裝 onnxruntime-gpu，速度可提升 10-20 倍。詳見 README.md 的 GPU 加速說明。
搜索 tag 找不到圖片？ ›
確認圖片已完成打標（左側顯示「未標記 0」）。若仍無法搜索，執行 python fix_tags.py 修復資料庫格式，再重啟應用程式。
如何完全重新打標？ ›
刪除 tags.db 和 thumbnails/ 資料夾，重新啟動後掃描資料夾，再點「開始打標」即可。
更換電腦後資料還在嗎？ ›
所有 tag 資料儲存在 tags.db，將整個 image-tagger 資料夾複製到新電腦即可，圖片路徑需相同才能正常顯示縮圖。
支援 macOS / Linux 嗎？ ›
後端程式碼理論上跨平台，但「在檔案總管中顯示」功能僅支援 Windows。macOS / Linux 用戶需將 start.bat 改為對應的 shell script。
